# How to Archive a Project

## Overview

Archiving removes inactive projects from active project views while preserving historical data.

## Steps

1. Open the project.
2. Select **Project Settings**.
3. Click **Archive Project**.
4. Confirm the action.

## Notes

* Archived projects remain accessible.
* Project data is preserved.
* Administrators can restore archived projects.

## Troubleshooting

If the archive option is unavailable, verify that you have administrator permissions.
