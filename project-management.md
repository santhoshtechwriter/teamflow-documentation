# Project Management

## Overview

Projects help teams organize work, track progress, and achieve goals efficiently.

Within TeamFlow, projects contain tasks, milestones, team assignments, and reporting data.

## Create a Project

To create a new project:

1. Sign in to TeamFlow.
2. Open your workspace.
3. Click **New Project**.
4. Enter a project name.
5. Add a project description.
6. Select project members.
7. Click **Create Project**.

The project will appear on the Projects dashboard.

## Create Tasks

Tasks represent individual units of work within a project.

To create a task:

1. Open a project.
2. Click **Add Task**.
3. Enter a task title.
4. Add a description.
5. Assign an owner.
6. Set a due date.
7. Click **Save**.

## Assign Team Members

Project managers can assign tasks to team members.

To assign a task:

1. Open the task.
2. Select **Assignee**.
3. Choose a team member.
4. Save the changes.

The assigned user will receive a notification.

## Update Task Status

TeamFlow supports the following task statuses:

| Status      | Description               |
| ----------- | ------------------------- |
| To Do       | Work has not started      |
| In Progress | Work is currently active  |
| Review      | Task is awaiting approval |
| Completed   | Task has been finished    |

## Track Project Progress

Project progress is automatically calculated based on completed tasks.

Project managers can:

* View completion percentages
* Monitor overdue tasks
* Track team workload
* Review project timelines

## Project Best Practices

To improve project outcomes:

* Create clear task descriptions.
* Assign ownership for every task.
* Use realistic deadlines.
* Review project progress regularly.
* Close completed tasks promptly.

## Troubleshooting

### Unable to Create a Project

Possible causes:

* Insufficient permissions
* Workspace limits reached
* Missing required fields

Resolution:

1. Verify your user role.
2. Check workspace limits.
3. Complete all required fields.
4. Try again.
